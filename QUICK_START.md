## üõ†Ô∏è Common Tasks

### Create a New Rule
1. Go to `/rules`
2. Click "New rule"
3. Fill in the form
4. Click "Create Rule"

### Add a Cluster
1. Go to `/clusters`
2. Click "New Cluster"
3. Enter cluster details
4. Click "Create Cluster"

### Track an Indicator
1. Go to `/indicators`
2. Click "New Indicator"
3. Enter indicator value
4. Set severity and confidence
5. Click "Create Indicator"

### Generate a Report
1. Go to `/reports`
2. Select report type
3. Choose time period
4. Select format
5. Click "Generate"

### View Notifications
1. Click bell icon in header
2. Or go to `/notifications`
3. Filter by type/status
4. Mark as read/archive

## üîß Troubleshooting

### Page Not Loading
1. Check console for errors
2. Verify route exists
3. Clear browser cache
4. Restart dev server

### Styles Not Applying
1. Check Tailwind classes
2. Verify component imports
3. Clear build cache
4. Restart dev server

### TypeScript Errors
1. Run `npm install`
2. Restart TypeScript server
3. Check for missing types
4. Verify imports
